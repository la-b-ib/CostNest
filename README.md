# <samp>CostNest

<samp>
  
**CostNest is a Chrome extension for expense tracking and financial analysis. It ensures security with multi-layer PIN authentication, encrypted local storage, and secure backup. Users gain intelligent expense categorization, real-time budget monitoring, multi-currency support, and exportable analytics through a responsive interface with interactive Chart.js  dashboards. Automation features include price detection, smart notifications, OCR receipt scanning, and browser integration.** 

<details>
  
**<summary>Features</summary>**

- **Security & Privacy: Multi-layer PIN authentication, encrypted local storage, compliance with data sovereignty, secure backup and restore.** 
- **Financial Management: Intelligent expense categorization, real-time budget monitoring, multi-currency support, detailed analytics with CSV export.**
- **User Experience: Material Design UI, responsive cross-device interface, interactive data visualization with Chart.js.**
- **Automation & Intelligence: Automated price detection, smart notifications, OCR-based receipt scanning, seamless browser integration.**
